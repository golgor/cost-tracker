# User Guide

Cost Tracker is a self-hosted household expense-sharing app designed for partners who share living
expenses. Unlike event-based splitters or budgeting tools, it's built for the continuous rhythm of
shared living: log as you go, settle monthly, transfer via bank, repeat.

## Key Features

- **Expense tracking** — Record shared expenses with flexible split options (even, percentage,
  shares, exact amounts)
- **Balance calculation** — Real-time balance showing who owes whom
- **Monthly settlements** — Review unsettled expenses together, confirm amounts, and record
  transfers
- **Recurring costs** — Define recurring expenses (rent, subscriptions) that auto-generate entries
- **Audit trail** — Full history of changes for transparency

## How It Works

1. **Log expenses** as they happen — quick capture on mobile, batch entry on desktop
2. **Review the balance** on the dashboard to see where things stand
3. **Settle monthly** — sit together, review expenses, confirm the settlement amount, and transfer
   via your bank
4. **Repeat** — the cycle resets after each settlement
