# Design Records

This section contains **design-phase artifacts** created during initial project planning. They are
preserved as historical records documenting the decisions and research that shaped Cost Tracker.

These are not living documentation — they reflect the state of thinking at the time of creation and
may not match current implementation in every detail. For current architecture decisions, see the
[Architecture](../architecture/index.md) section.
